## ⭐ How Scoring Works

Scores reflect Discord reactions (emojis) in `#base-guild-showcase`. Here the details:

| Source | Points | Reason |
|---|---|---|
| ⭐ reaction | +2 pts each | Top tier base |
| 👍 reaction | +1 pt each | Good build, must-visit |
