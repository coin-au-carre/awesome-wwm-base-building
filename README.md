## ⭐ How Scoring Works

Scores reflect Discord reactions in `#base-guild-showcase`, weighted by how broadly each person votes.

| Source | Points |
|---|---|
| ⭐ reaction | +2 pts per qualifying vote |
| 👍 reaction | +1 pt per qualifying vote |
| 🔥 reaction | +1 pt per qualifying vote |
| 📜 Lore filled | +1 pt |
| 🗺️ What to visit filled | +1 pt |

**Votes are weighted by breadth** - to keep things fair and discourage guilds from just rallying their own members to vote only for themselves:

| Guilds reacted on | Vote weight |
|---|---|
| 1 guild | ✗ not counted |
| 2–5 guilds | ×1 |
| 6–9 guilds | ×2 |
| 9+ guilds | ×3 |

The more guild bases you explore and react on, the more your votes count. If you only react on your own guild, they don't count at all.
